# 🛍️ Paradise Store

Une boutique e-commerce moderne pour les téléphones (iPhone, Android) et les PC portables.

## 🌟 Fonctionnalités

- 📱 Catalogue de produits (iPhone, Android, PC Portables)
- 🛒 Panier d'achat fonctionnel
- 🌙 Mode sombre
- 🔍 Recherche en temps réel
- 📱 Design responsive (mobile, tablette, desktop)
- 🌍 Support multilingue (FR, EN, ES)
- 💬 Formulaire de support client

## 🚀 Démarrer

1. Clonez le dépôt
```bash
git clone https://github.com/iamaxisdrama-glitch/paradise-store.git
```

2. Ouvrez `index.html` dans votre navigateur
